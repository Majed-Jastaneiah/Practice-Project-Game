# Dodge Master - Claude Briefing

## What is this project?

Dodge Master is an infinite survival mobile game built with
React Native and Expo. Players control a glowing gold dot
and dodge incoming obstacles that increase in speed over time.

## Developer

Majed - solo developer. Explain every step clearly and simply.

## Tech Stack

- React Native / Expo (create-expo-app)
- EAS CLI for building and publishing
- Node.js / npm
- Git / GitHub (repo: Practice-Project-Game)
- Backend: TBD (Firebase or Supabase)

## Project Structure

- /app → screens and navigation
- /components → reusable components
- /hooks → custom hooks
- /constants → colors, config, coin values
- /assets → images and fonts
- /utils → helper functions
- /services → API and backend calls

## Key Rules

- Never hardcode secrets, API keys, or coin values
- Always use .env for sensitive config
- Always consider iOS and Android
- Keep code modular and reusable
- Clean, well commented code
- Error handling on everything
- Commit regularly to Git

## Coin Economy

- New players get 30 free coins on registration
- Coins used to revive in game
- Coins earned via milestones, daily login, watching ads
- Coins purchased via IAP
- Never hardcode coin values — always use constants
